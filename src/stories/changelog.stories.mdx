# Changelog

## 0.0.19

24.12.2020<br/><br/>

#### Features

1. Tooltip: New props `hasArrow`

### Bugfixes

1. Tooltip: Hide scroller if there's enough height

## 0.0.18

23.12.2020<br/><br/>

#### Features

1. Tooltip: new props `hideDelay`, `showDelay`
1. Tooltip: new prop `maxHeight` which allows you to create scrollable tooltips
1. Tooltip: tooltip will switch position if there's no room to show it

## 0.0.17

22.12.2020<br/><br/>

#### Breaking changes

1. Dropdown: slot `options` is now the default slot
1. Zen-option: slot `label` is now the default slot

#### Features

1. Dropdown: Added slot `placeholder`
1. Dropdown: Added prop `disabled`
1. Dropdown: Selected item is copied into the field, so the item in the field is styled correctly (has the same content as an original item - icons, padding & stuff)
1. Zen-option: Added prop `disabled`

#### Bugfixes

1. Dropdown: prevent closing dropdown on scrolling using the scrollbar
1. Dropdown: The height of the field should follow the height of `<zen-dropdown>` tag

## 0.0.18

#### Breaking changes

1. zen-spinner: **Prop `color` removed**. To set custom color: `<zen-spinner style="color: #f00">`
1. zen-button: **Prop `label` removed**. To set button label: `<zen-button>Click here</zen-button>`
