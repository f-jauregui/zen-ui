# Changelog

## 0.0.17

22.12.2020<br/><br/>

#### Breaking changes

1. Dropdown: slot `options` is now default slot
1. Zen-option: slot `label` is now default slot

#### Features

1. Dropdown: Added slot `placeholder`
1. Dropdown: Added prop `disabled`
1. Dropdown: Selected item is copied into field, so item in field is styled correctly (has the same content as original item - icons, padding & stuff)
1. Zen-option: Added prop `disabled`

#### Bugfixes

1. Dropdown: prevent closing dropdown on scrolling using scrollbar
1. Dropdown: Height of the field should follow the height of `<zen-dropdown>` tag
