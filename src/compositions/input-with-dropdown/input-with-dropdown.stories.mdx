# Dropdown

This component **is NOT really a component**. It's actually just a composition of `Input` and a `Dropdown` component. Dropdown is placed into input using input's slot `trailingSlot`.

## Component preview

export const Template = ({ options, menuWidth, fieldAlign }) => {
  return html`<zen-input placeholder="Username" class="mb-80">
    <zen-dropdown slot="trailingSlot" borderless="true" placeholder="choose role" style="max-width: 300px;" .options=${options}>
      <div slot="options">
        ${options.map((item, index) =>
          html`<zen-option label=${item.label} value=${item.label}></zen-option>`,
        )}
      </div>
    </zen-dropdown>
  </zen-input>`;
}
