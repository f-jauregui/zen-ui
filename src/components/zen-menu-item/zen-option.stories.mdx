# Zen Option

Option item to be used inside **zen-dropdown** component

## Component previews

<br />

### 1. Sizes

- Zen-option ships in different sizes
- These sizes only matter if you have default items (simple text).
- Make sure you also set the same `size` value to zen-dropdown!
